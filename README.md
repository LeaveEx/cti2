



#### Deploy in your VPS
````bash
git clone https://github.com/muhammadrizky16/MenfessBot/
cd MenfessBot
pip3 install -r requirements.txt
nano config.py
python3 main.py
````

#### START_MESSAGE | FORCE_SUB_MESSAGE

* `{first_name}` - User first name
* `{last_name}` - User last name
* `{id}` - User ID
* `{mention}` - Mention the user
* `{username}` - Username
* `{fullname}` - Username
